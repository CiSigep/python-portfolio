Musician's Keeper

Links: Repository Heroku
Description:

An expense tracker to help self-employed musicians keep track of their expenses in preparation for tax day. A user can login and log expenses that they can write off on their taxes. They add the date, name, and amount to the category the expense falls in. They can then add a picture of the receipt as proof of their purchases, or delete the expense if they made a mistake. Uses React, NodeJS, Express, Sequelize, PassportJS, JWTs, and Material-UI.

CareMinder

Links: Repository Heroku
Description:

An app for caregivers to keep track of the ones they are taking care of. Caregivers can log in to the app to access all the people they are taking care of. After selecting one, they will be take to that patient's calendar, where they can add tasks, which can cover anything from doctor's appointments to medicine taking, or bills, which cover any of their monetary transactions. Uses NodeJS, Express, HandleBars, Sequelize, PassportJS, Bootstrap, jQuery, and FullCalendar.

Magic Carpet

Links: Repository Pages
Description:

Welcome aboard the Magic Carpet! A Web Application that aggregates travel information for a travelers viewing. It allows the user to view flight quotes between cities, see events in said cities, and if they're going quickly, some weather in the form of a 5-day forecast. Users can click on things in the displayed information to highlight them and click save to save their work, or clear to start the whole process again. Uses Bootstrap, jQuery, jQuery Validator, HTML, and CSS.

Google Books Search

Links: Repository Heroku
Description:

A books search app that searches for books using the Google Books API. Users can search for and save books for later viewing. The app has a view button so the user can go look at the book in the Google Books app. Uses Bootstrap, HTML5, React, NodeJS, Express, and MongoDB.

MongoScraper

Links: Repository Heroku
Description:

This app is a news article scraping app that grabs articles from GlobeNewswire and displays them on its page. The user can save the articles and view them on the saved articles page. Once there they can add notes to the saved articles. This application is a NodeJS app that uses cheerio to scrape, MongoDB for data storage, and handlebars to display the pages.
